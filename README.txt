# After setting up the database, displaying the results in Unity

Team Project (5 members)(2024):
    My contributions - Documentation Specialist, Technical Lead
    ER Diagram, database implementation, and displaying results in Unity. We worked together until the database implementation and decided how to display the results. The other members chose to develop a website.
    
Project Concept:
    After learning about the archer's information, we created personas and developed individual user stories. Based on these, we designed an ER diagram, made necessary revisions, and built the database. Then, we developed a program that aligns with the user stories.

Tools and Services:
    MySQL - Relational database management system for user and product management
    Unity - The game engine used to create ArcherSystem
    Visual Studio Code - Used to edit scripts

    Confluence - Used for documentation and collaboration within the team
    Jira - Used for project management and issue tracking
    Microsoft Teams - Used for team communication
    Canva - Used for creating presentation
